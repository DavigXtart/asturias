# Asturias 2026 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build "Asturias 2026", a mobile-first web app to organize a rural-house trip for ~21 friends: registration/attendance, carpooling, a random costume-pairing draw, and per-day room distribution.

**Architecture:** Three tiers. A React + Vite + Tailwind SPA (Render Static Site) talks to a Spring Boot REST API (Render Web Service), which persists to a Supabase Postgres database via JDBC/JPA. No Supabase Auth/Realtime. Identity is name-based (no passwords); admin actions are gated by a PIN. The rooms screen refreshes via 5s polling.

**Tech Stack:** Java 21, Spring Boot 3.3, Maven, Spring Web, Spring Data JPA, Flyway, PostgreSQL, Bean Validation, JUnit 5 + Testcontainers. React 18 + TypeScript, Vite, TailwindCSS, React Router, TanStack Query, Axios, Framer Motion, dnd-kit, Vitest + React Testing Library. All frontend work follows the `ui-ux-pro-max` skill (modern, elegant, SaaS, maximally dynamic).

**Spec:** `docs/superpowers/specs/2026-06-06-asturias-design.md`

---

## Conventions

- **Repo layout:** monorepo with `backend/` (Spring Boot) and `frontend/` (React).
- **Identity headers:** the SPA sends `X-Guest-Id: <uuid>` for "who am I" actions and `X-Admin-Pin: <pin>` for admin actions. The backend trusts `X-Guest-Id` (no passwords, by design) and validates `X-Admin-Pin` against `app_config`.
- **IDs:** UUID primary keys generated by the app.
- **Dates:** `LocalDate` everywhere (trip days). Trip window default 2026-07-10 → 2026-07-17.
- **Commits:** conventional commits, one per task step where indicated.
- **Run backend tests:** `cd backend && ./mvnw test`. **Run one test:** `./mvnw test -Dtest=ClassName#method`.
- **Run frontend tests:** `cd frontend && npm test`.
- **TDD:** write the failing test, see it fail, implement minimally, see it pass, commit.

```java
package com.asturias2026.common;
import com.asturias2026.common.dto.ApiError;
import org.springframework.http.*;
import org.springframework.web.bind.MethodArgumentNotValidException;
import org.springframework.web.bind.annotation.*;

@RestControllerAdvice
public class ApiExceptionHandler {
    @ExceptionHandler(ApiException.class)
    public ResponseEntity<ApiError> handleApi(ApiException e){
        return ResponseEntity.status(e.status()).body(new ApiError(e.getMessage()));
    }
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public ResponseEntity<ApiError> handleValidation(MethodArgumentNotValidException e){
        String msg = e.getBindingResult().getFieldErrors().stream()
            .map(f -> f.getField()+": "+f.getDefaultMessage()).findFirst().orElse("Datos inválidos");
        return ResponseEntity.badRequest().body(new ApiError(msg));
    }
}
```

```java
// AdminPinInterceptor.java
package com.asturias2026.config;
import com.asturias2026.config_.ConfigService;
import jakarta.servlet.http.*;
import org.springframework.web.servlet.HandlerInterceptor;

public class AdminPinInterceptor implements HandlerInterceptor {
    private final ConfigService config;
    public AdminPinInterceptor(ConfigService config){ this.config = config; }
    @Override public boolean preHandle(HttpServletRequest req, HttpServletResponse res, Object handler) {
        if (config.pinMatches(req.getHeader("X-Admin-Pin"))) return true;
        res.setStatus(403);
        return false;
    }
}
```

```java
    var palette = List.of("rojo","azul","verde","amarillo","morado","naranja","rosa","cian","lima","fucsia");
    for (int i=0;i<groups.size();i++){
        var pairEntity = pairRepo.save(new CostumePair(draw.getId(), i, palette.get(i % palette.size())));
        for (var gid : groups.get(i)) memberRepo.save(new CostumePairMember(pairEntity.getId(), gid));
    }
    return new DrawResult(draw.getId(), groups.size());
}
public MyPairResponse myPair(UUID guestId){
    var pair = pairRepo.findPairOfGuest(guestId)
        .orElseThrow(() -> notFound("Aún no tienes pareja (¿se ha hecho el sorteo?)"));
    var partners = memberRepo.findGuestNamesByPairExcluding(pair.getId(), guestId); // List<String>
    return new MyPairResponse(pair.getBallColor(), partners);
}
```
